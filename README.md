FitFusion
FitFusion is a comprehensive web-based platform designed to bridge the gap between physical fitness management and nutritional e-commerce. Built with the MERN stack, it provides users with a seamless experience for managing memberships, tracking fitness goals, and purchasing high-quality supplements.

🚀 Key Features
User Authentication: Secure registration and login with email verification to ensure account authenticity.

Fitness Management: Tools for users to manage their gym memberships and view personalized diet plans.

Integrated Shop: A fully functional e-commerce module for browsing and purchasing health supplements.

Multi-Gateway Payments: Integrated support for local payment gateways like eSewa and Khalti for seamless transactions.

Real-time Communication: An Admin-User messaging hub allowing members to communicate directly with trainers.

Admin Dashboard: A centralized control panel for administrators to manage products, view sales statistics, and oversee member inquiries.

🛠 Tech Stack
Frontend: React.js, React Router, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB, Mongoose

Authentication: JWT (JSON Web Tokens)

Payment Integration: eSewa API, Khalti API

Deployment Tools: Environment Variables (dotenv), CORS for secure cross-origin requests

🏗 Project Architecture
Client: React-based SPA (Single Page Application) for dynamic user interaction.

Server: RESTful API built with Express, organized into modular route handlers.

Database: NoSQL storage using MongoDB to manage relationships between users, products, and messages.
